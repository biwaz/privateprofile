# privateprofile

These are functions to read/write ini-file with vbs.

Both GetPrivateProfile / WritePrivateProfile of this module correspond to quart processing by " ' equivalent to GetPrivateProfileString of Win32 API.
Since I will publish the source, please omit unnecessary processing.

The appearance of powershell brings that VBS has gone with an out-of-date feeling, but I am pleased if we can extend the life even a little.

Please send us your comments / impressions.
